Place your app icons here:

  app.ico    — Main application icon (16, 32, 48, 256 px multi-size ICO)
  app.png    — PNG version for About page (80x80 px recommended)
  tray.ico   — System tray icon (16x16 and 32x32 px, transparent background)

You can convert the macOS .icns assets from NestVault_Xcode using:
  - macOS: iconutil or sips
  - Windows: IcoFX, GIMP, or online converters
  - Minimum: 16x16, 32x32, 48x48, 256x256 in the .ico file

The tray icon should work well at 16x16 — keep it simple and recognizable at small sizes.
